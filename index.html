  • Code: VBA/SQL
  • About: Design a cricket simulating game on Microsoft Excel and use MicrosoftAccess to store all the game stats
  • Design:
    • The game will consist of a couple of buttons which randomly generate numbers between 0-6 (runs scored) through VBA coding. If the number generated is 5 it will automatically generate another random number between 1-10 (BowlersRoll) through VBA coding. Using Microsoft Access as a type of server, SQL queries and VBA coding have been created to determine players and teams stats, such as batting averages and strike rates, bowlers average and economies, team stats and many more.
